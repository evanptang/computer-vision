Sudoku-Read Me
By Evan Tang

This project allows the player to input any sudoku board and to receive a solved
solution.

It uses three independent ways to reach that solved solution, Backtracking,
Forwardchecking and through minimum remaining value.

For further information on the files see below
  -sudoku.py: executes the actual commands. Server interaction starts at line 94
  -all_functions.py: the definitions for the functions utilizing sudoku
  algorithms as well as their helper functions.
  -common.py: shared definitions.
